This is a demo rest app who is simulating twitter.In this project I used Java SE&EE,MySQL,Spring.Functionalities:
-register a user
-searching a user
-following another user
-adding a post for that user or other users
-liking a post
-reply to a post
-getting my own posts
-getting feed(the posts for the users that the current user is following)
I tested all this functionalities using PostMan
